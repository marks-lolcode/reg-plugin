Registration Check-In

Toolbar face = Reggie.
No blue "MERCH MODE" banner.
If you see MERCH MODE, tell your lead.

What the icon color means

BlueGood — issue badge
YellowRead the note first
RedSend to Help Desk
GreyNot on a check-in page

Steps

  1. Find the attendee. Search in Neon, click their name to open the account page. The check-in window opens on its own (or click Reggie).
  2. Read the account screen. Green "Ready" → wait, it moves you forward. Yellow note → read it, tick the box, click Proceed to Check-In →. Red → SEND TO HELP DESK.
  3. Pick the attendee. Click Check In → on their row. A row that says "Send to Help Desk" is blocked.
  4. Do what the screen asks. Yellow warning → Show me the field ↓, fix it, click Re-check ↺.
    Age check (Adult & Dealer badges only)
    1. Screen shows: Legal Name + "DOB on or before [date]"
    2. Ask for a government-issued photo ID (driver's license, passport, or state ID card).
    3. Confirm the face on the ID matches the person in front of you.
    4. Check the date of birth on the ID — it must be on or before the date shown on screen.
    5. Return the ID immediately.
    6. Click Age Verified, ID Returned ✓
    7. If DOB is too recent, face doesn't match, or no valid ID → call a manager.
  5. Issue the badge. When the BADGE NUMBER shows, click Badge Issued. If it says Badge Issued - Send to Merchandise, tell them to visit the Merch table.
[SCREENSHOT: ready-to-issue check-in screen with BADGE NUMBER]

Call a manager / Help Desk if…

  • Anything red, or any "Send to Help Desk" / "NOT ALLOWED".
  • A hold (Registration, Art Show, or Operations).
  • ID name doesn't match, or badge was already issued.
  • The box says contact the Registration Head, or the badge file didn't download.
When in doubt — don't issue the badge. Ask.
