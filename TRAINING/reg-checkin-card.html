Registration Check-In

The pop-up guides you — just follow along.

Steps — when everything goes well

  1. Neon searchFind the attendee — ask for their ID. Search Neon, click their name. Ask for a government photo ID now. The pop-up opens and moves you forward when all-clear (green).
  2. Reg-list pop-upPick the attendee. Click Check In → on their row.
  3. Check-in pop-upVerify age when it comes up. Adult & Dealer badges show ID Required. If over 18, verify the ID below, then Age Verified, ID Returned ✓.
    Age check (Adult & Dealer badges only)
    1. Screen shows Legal Name + "DOB on or before [date]".
    2. Use the photo ID you took at the start.
    3. Name on the pop-up must match the name on the ID.
    4. Face on the ID matches the person.
    5. Date of birth is on or before the date shown.
    6. Return the ID, then click Age Verified, ID Returned ✓.
    7. Name mismatch, no ID, or unsure → get the Room Captain or Help Desk.
  4. Check-in pop-upIssue the badge.
    1. Pre-Printed / Printed? / Blank (read first — runner gets the badge): Pre-Printed = pull from stock · Printed? = check stock · Blank = print one.
    2. First Time? Badge Ribbon! → give a First Time Attendee ribbon.
    3. Get the badge, hand it over, then click Badge Issued. If it also says Send to Merchandise, send them to the Merch table.

Send to Help Desk

  • The pop-up tells you to (Send to Help Desk / NOT ALLOWED).
  • No photo ID.
  • Under 16 and alone.

Call the Room Captain

  • ID verification issues — name/face mismatch or you're unsure.
  • Any question you're not sure about.
When in doubt — don't issue the badge. Ask.
